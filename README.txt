MindIT is a program to create and edit database of employees of your company. Use appropriate tools to add, update, delete or simply review your employees’ database. Create a report for official and personal needs. To create report, fill in the ReportName field and choose the type of report.
All necessary files are to be found in this folders. All plugins for successful operation including PostgreSQL Driver, ITEXT and JavaFX SDK 11.0.2 are stored in lib folder. 
All source files packages are placed in employeeDatabase package.

To create MindIT, I have used JDK 11.0.1

To login into MindIT for the first time, type in “root” in Username field and Password field. For the next time, proceed with your Department name as Username and Surname as a Password.
